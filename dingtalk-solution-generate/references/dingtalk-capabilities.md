## 钉钉DEAP平台能力详解

| 能力 | 说明 | 方案引用场景 |
| --- | --- | --- |
| 知识库检索 | 基于企业知识库的语义搜索和问答 | 智能客服回答产品/服务/流程问题 |
| 工作流编排 | 多步骤AI任务的可视化编排 | 内容生成：热点分析→选题确认→初稿生成→人工审核 |
| MCP插件 | 标准协议接入外部API和系统 | 对接企业微信、小红书、舆情系统、HIS |
| 多模型接入 | 支持通义千问等模型，可切换 | 不同场景选用不同模型（客服用高性价比、内容生成用高质量） |
| 评测体系 | 智能体效果评测和持续优化 | 上线前评测达标、上线后监测优化 |
| 权限管控 | 按组织架构控制知识库和智能体访问权限 | 不同门店的客服智能体只能看到本店知识 |
| 智能体模板 | 预置行业和角色模板，快速搭建 | 售后客服模板开箱即用，在模板基础上定制业务知识 |
